# VOA multiplayer start (do NOT use vanilla New Game)

With Skyrim Platform + skymp5-client installed:

1. Launch via SKSE (VOA launcher Play)
2. Stay on MAIN MENU (or press ~ and wait for console: Hello Multiplayer)
3. Client auto-connects to server from settings
4. Multiplayer spawn uses loadGame — Helgen is skipped

If still stuck on main menu without connect:
- Press F2 for browser UI
- Check console (~) for Connection failed

Emergency manual skip Helgen (singleplayer only, not for MP):
- At main menu: ~ then: coc Riverwood
