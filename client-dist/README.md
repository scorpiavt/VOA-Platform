# Visions of Aetherius — Client Files (optional manual install)

Players should use the public launcher (`VOA-Public\VisionsOfAetherius.exe`), which installs the client automatically on Play.

## Manual path (fallback only)

Copy into:

```
<Skyrim SE>\Data\Platform\Plugins\
```

| File | Purpose |
|------|---------|
| `skymp5-client.js` | SkyMP multiplayer client |
| `skymp5-client-settings.txt` | Written by the launcher on Play |

## Official server

- IP: `178.156.158.116`
- Port: `10000`

## Do not use

- Keizaal client bundles / activity plugins
- Developer monorepo paths
